# FMSE

## Installation
```
npm install -g ionic cordova
```

### Build application
```
npm run build
```

## Docker

### Build docker container
```
npm run docker:build
```

### Start docker
```
npm run docker:start
```

#### Stop docker
```
npm run docker:stop
```

#### Visit url
```
localhost:8080
```

## Development
```
npm run ionic:serve
```
